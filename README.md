# Instituto — Site estático

Os arquivos do site estão em `stitch_terapeuta/` (originais/exportados).  
A explicação do projeto está em `instituto_resumo.txt`.

## Como visualizar o site

Abra no navegador:

- `site/index.html`

## Estrutura

- `site/`: versão organizada para navegação (links entre páginas funcionando)
- `stitch_terapeuta/`: páginas originais (cada uma em uma pasta com `code.html`)
